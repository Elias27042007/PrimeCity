Diese Ressource ist ein Platzhalterordner.

Installiere oxmysql aus der offiziellen Quelle:
- Repository: https://github.com/overextended/oxmysql

Zielordner:
server/resources/[dependencies]/oxmysql

Wichtig:
- Resource-Name muss `oxmysql` sein.
- Stelle sicher, dass der Ordner `lib/MySQL.lua` enthalten ist.
