Diese Ressource ist ein Platzhalterordner.

Installiere hier ESX Legacy Core aus der offiziellen Quelle:
- Repository: https://github.com/esx-framework/esx_core
- Empfohlener Stand: aktueller stabiler Release-Tag (z. B. 1.13.x)

Zielordner:
server/resources/[dependencies]/es_extended

Wichtig:
- Stelle sicher, dass der Resource-Name exakt `es_extended` ist.
- Falls aus esx_core mehrere Ressourcen benoetigt werden, platziere sie strukturiert unter [dependencies] und passe server.cfg bei Bedarf an.
